# Magnetjakten

Magnetjakten är ett litet webspel för barn där man styr en magnet, samlar stjärnor och följer med genom tre mjuka och färgglada världar tillsammans med en glad enhörning.

Spelet är gjort för att kännas lugnt, enkelt och roligt att spela på mobil. All text i spelet är på svenska och gränssnittet är byggt för stora tryckytor och touchkontroller som är lätta att använda.

## Så funkar spelet

- Tryck på `Spela` för att börja.
- Välj `Lätt` eller `Utmaning` innan du startar.
- Dra magneten med fingret eller musen för att samla stjärnor.
- När du samlar en stjärna blir det små gnistor och ett litet ljud om ljudet är på.
- När en nivå är klar går spelet vidare till nästa värld.

## Världar

Spelet innehåller tre nivåer i följd:

1. `Magnetängen`
2. `Regnbågsskogen`
3. `Glittergrottan`

Varje värld har sin egen färgkänsla och sitt eget lilla tema, men reglerna är enkla så att barnet snabbt känner igen sig.

## Svårighetslägen

### Lätt

- Starkare magnetkänsla
- Mer förlåtande samling
- Lugnare tempo

### Utmaning

- Lite mindre hjälp från magneten
- Något mer krav på timing
- Fortfarande barnvänligt och utan stress

## Ljud

Spelet har små ljudeffekter som passar med samlandet och vinsten. Ljudet kan stängas av direkt i spelet.

## Starta spelet

Kör den lokala servern med:

```bash
npm start
```

Öppna sedan adressen som visas i terminalen, till exempel `http://localhost:3001/`.

## Teknik

## Mål med spelet

Magnetjakten är byggt för att vara:

- lätt att förstå
- tryggt och vänligt
- lagom utmanande
- fint att spela på mobil
- enkelt att bygga vidare på i nästa version
